# pomodoro-js
바닐라 자바스크립트로 구현한 SPA 포모도로 타이머

### 사용 기술
- HTML/CSS/Javascript
- Webpack
- Jest

### 구현 내용
- [클래스 컴포넌트](https://dev-hkim.tistory.com/15)
- Diffing을 통한 렌더링
- HistoryApi 기반의 라우팅
- 전역 상태관리 store
